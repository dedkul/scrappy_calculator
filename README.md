# scrappy_calculator — iOS App

A playful, scrapbook-themed calculator app I built as a **vibe coding** experiment — using mostly AI tools from idea → design → shipped UI.

---

![scrappy_calculator demo](scrappy_calculator/Instructions/scrappy_calculator.gif)

## The VibeCoding Process (Designer-first)

This was my **second** vibe-coding project. The goal was simple: see what I can create using **AI-first design + AI-assisted coding**.

- **Design (Variant UI / AI)**: I explored a scrapbook-style calculator concept in Variant and used that as the visual north star.  
  - Variant link: [Variant shared design](https://variant.com/shared/26153c5b-b2e3-4df5-befa-e671ec469dbf?t=1773899721232)
- **Build (Cursor)**: I built the app in Xcode, but did most of the implementation by working inside **Cursor** and iterating quickly with AI help.
- **Iteration loop**: tweak the design vibe → update UI → polish spacing/typography → repeat until it felt right.

## Features

- A scrapbook-y calculator UI (paper display, doodles, press animations)
- Basic operations: **+**, **−**, **×**, **÷**
- **AC**, **del**, **%**, and **.**
- Works offline (no internet needed)

## Run It

1. Open `scrappy_calculator.xcodeproj` in Xcode
2. Run the `scrappy_calculator` scheme on a simulator or device
